# JACOBI BENCHMARKING

## HOW TO COMPILE

```bash
$>make jacobi    /* Serial Version */
$>make jacobi_mp /* openMP Version */
```

## HOW TO RUN

```bash
$>./jacobi -n <size> -i <iterations> -c <convergence> /* Serial Version */
$> openMP will be updated soon.
```

## HOW TO TEST

```bash
$>make test    /* Serial Version */
$>make test_mp /* openMP Version */
```
